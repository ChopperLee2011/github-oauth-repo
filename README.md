oauth-github-repo
========
## install

```javascript
bower install;
npm install;
```

## run
```javascript
node bin/www
```

## work flow
1. bowser http://localhost:3000/
2. login with github oauth
3. get personal repo list
4. click repo link
5. get issue list
6. click issue link
7. get issue detail
